# ResumeMatch AI

ResumeMatch AI is a full-stack web application that compares resumes to job descriptions and provides feedback using keyword matching.

## Features

- Resume/job description analysis
- Match score calculation
- Matched and missing skills
- MongoDB database integration
- Saved analysis history
- Responsive frontend UI

## Technologies Used

Frontend:
- React
- Vite

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas
- Mongoose

## How to Run

### Frontend

cd frontend
npm install
npm run dev

### Backend

cd backend
npm install
node server.js
